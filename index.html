Acerca del equipo

¿A qué nos dedicamos?

Somos estudiantes de 4to nivel de la carrera de Ingeniería en Sistemas y Gestión de Data en el Tecnológico Universitario Rumiñahui. Nos encontramos cursando la asignatura de Integración y Reutilización de Software.

Nuestro equipo trabaja en el diseño, desarrollo y documentación de soluciones de software, aplicando metodologías modernas de versionamiento y colaboración como Git y plataformas remotas como GitHub.

En este proyecto demostramos el uso práctico de Git como versionador de código fuente, integrando herramientas de Visual Studio .NET para el desarrollo colaborativo real.
